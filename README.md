# Trolls and Games 🧌🎮

Welcome to **Trolls and Games** — a fun repository full of **mini games, harmless troll scripts, and programming experiments**.

This project is made for **learning, entertainment, and experimenting with simple games and funny code**.

---

## 🎮 Play Games

This repository includes small games that can be played directly from your computer or browser.

Examples may include:

* Guess the Number
* Tic-Tac-Toe
* Snake
* Terminal-based games
* Simple browser games

These games are lightweight and easy to run.

---

## 🧌 Troll Scripts

The **trolls** section contains harmless prank scripts made for fun.

Examples:

* Fake loading screens
* Funny terminal spam
* Fake virus simulations
* Silly developer jokes

These scripts are meant to **entertain and demonstrate programming concepts**.
